# PollsAppBlazor

Simple Polls Web App built on ASP.NET Core and Blazor WebAssembly.

## Tech Stack

### Front End
* MudBlazor

### Back end
* Duende IdentityServer
* Entity Framework
* Swashbuckle
* SQL Server
