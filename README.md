# PollsAppBlazor

Simple Polls Web App built on ASP.NET Core and Blazor WebAssembly. This project is currently in active development.

## Tech Stack

### Front End
* MudBlazor

### Back end
* Duende IdentityServer
* Entity Framework
* Swashbuckle
* SQL Server
